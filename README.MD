## Chức năng đã làm

- Thiết kế giao diện website giới thiệu khóa học trực tuyến (EduMaster) theo hướng responsive bằng Bootstrap 5.
- Xây dựng thanh điều hướng (Navbar) cố định với hiệu ứng cuộn mượt đến từng phần của trang.
- Thiết kế Hero Section với nút kêu gọi hành động (Call to Action).
- Xây dựng phần giới thiệu (About) với các thẻ thông tin và biểu tượng minh họa.
- Hiển thị danh sách các khóa học bằng Card Bootstrap.
- Tạo trang chi tiết cho từng khóa học: HTML, CSS, JavaScript và Bootstrap.
- Hiển thị thông tin giảng viên và các kỹ năng chuyên môn.
- Xây dựng biểu mẫu liên hệ (Contact Form).
- Kiểm tra dữ liệu biểu mẫu bằng JavaScript:
  - Kiểm tra họ tên.
  - Kiểm tra định dạng email.
  - Kiểm tra số điện thoại.
  - Kiểm tra nội dung liên hệ.
- Hiển thị thông báo khi gửi biểu mẫu thành công.
- Tích hợp chức năng Dark Mode và lưu trạng thái bằng LocalStorage.
- Tạo nút "Back to Top" giúp quay lại đầu trang với hiệu ứng cuộn mượt.
- Thêm hiệu ứng Scroll Animation bằng Intersection Observer.
- Xây dựng chức năng tìm kiếm khóa học theo tên bằng JavaScript.
- Tạo khu vực FAQ với chức năng đóng/mở nội dung.
- Thêm hiệu ứng Loading khi trang được tải hoàn tất.
- Tối ưu giao diện trên thiết bị Desktop, Tablet và Mobile.
- Tổ chức mã nguồn theo cấu trúc thư mục rõ ràng gồm HTML, CSS và JavaScript.
